\# Audio Converter \& Extractor (音声抽出・変換ツール)



動画ファイルから音声を抽出したり、オーディオファイルの形式を変換したりするためのWebツールです。

ブラウザ上で全ての処理が完結するため、大容量の動画ファイルをサーバーにアップロードする待ち時間がなく、プライバシーも安全です。



\## 🌟 主な機能



\* \*\*動画からの音声抽出\*\*: MP4, MOV, WebM などの動画ファイルをドラッグ＆ドロップするだけで音声のみを取り出せます。

\* \*\*出力形式の選択\*\*:

\* \*\*WAV\*\*: 無圧縮の高音質ファイル。音質を維持したい場合に最適です。

\* \*\*MP3\*\*: 192kbpsの圧縮ファイル。ファイル容量を抑えたい場合に最適です。





\* \*\*一括ダウンロード\*\*:

\* 処理した全ファイルを個別に一括保存。

\* 全ファイルを1つの \*\*ZIP形式\*\* にまとめて保存。





\* \*\*100% ローカル処理\*\*: データの送信は一切行われません。オフライン環境でも動作可能です。



\## 🛠 技術仕様



\* \*\*対応入力\*\*: MP4, MOV, WebM, AVI, MP3, WAV, FLAC, AAC 等

\* \*\*出力形式\*\*:

\* WAV (16bit PCM)

\* MP3 (192kbps / Lameエンコーダー使用)





\* \*\*コア技術\*\*: Web Audio API を使用してデコードとリサンプリングを行い、JavaScriptでバイナリを構築しています。



\## 🚀 使い方



1\. 上部のセレクトボックスから、出力したい形式（WAV または MP3）を選択します。

2\. 変換したい動画・音声ファイルを枠内にドラッグ＆ドロップします。

3\. 変換完了後、リストから「保存」ボタン、または一括ダウンロードボタンをクリックしてください。



\## ⚖️ ライセンス



本ツールは \*\*LGPL\*\*（lamejsに準拠）となります。詳細はリポジトリのライセンス規定を参照してください。

\### 注意事項



\* \*\*MP3変換の負荷\*\*: MP3へのエンコードはブラウザのCPUを使用するため、非常に長い動画ファイルを処理する場合、完了まで時間がかかることがあります。

\* \*\*ブラウザの許可\*\*: 「全ファイルを個別に保存」を使用する際、ブラウザから「複数ファイルのダウンロード許可」を求められた場合は「許可」を選択してください。
