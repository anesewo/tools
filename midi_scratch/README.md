# MIDI → 簡易楽譜変換ツール (High-Precision Edition)

MIDIファイルを読み込み、特定のテキスト形式（音高・音長のリスト）に変換するWebツールです。
GitHub Pagesでそのまま動作し、サーバーサイドの処理は一切不要です。

## 🚀 特徴

- **高精度な変換**: 内部補正ロジックを排除し、純粋にMIDIのTick数を拍数に変換。
- **ズレの最小化**: 小数第4位までの高精度出力により、三連符などの割り切れないリズムでも長時間の同期を維持。
- **和音の自動分離**: 複数の音が重なっている場合、自動的に「piano1」「piano2」...とパートを分けて出力します。
- **ゼロ除外**: 変換時に発生する待機時間「0」などの不要なデータは自動的にフィルタリングされます。
- **オフライン動作**: ブラウザ上で全処理が完結するため、アップロードしたMIDIデータが外部に送信されることはありません。

## 📋 使い方

1. `index.html`（または公開URL）をブラウザで開きます。
2. **ファイルを選択** ボタンから、変換したいMIDIファイル（.mid）を選択します。
3. **変換開始** ボタンをクリックします。
4. 画面下のエリアに変換結果が表示されます。
5. **txtを保存** ボタンで結果をテキストファイルとしてダウンロードできます。

## 🎵 出力フォーマットの仕様

出力されるテキストは以下の順序で構成されます：

1. パート名（例: `piano1`）
2. 休符がある場合：`（スペース）` + `拍数`
3. 音高（例: `a`, `w1`, `d2` など）
4. 音長（例: `0.5`, `0.3333` など）
5. 最後に `last` という文字列で終了

## 🛠 技術スタック

- **Core**: JavaScript (ES6+)
- **Library**: [@tonejs/midi](https://github.com/Tonejs/Midi) (MIDIファイルの解析に使用)
- **UI**: Pure HTML / CSS
