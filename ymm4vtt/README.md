# ymm4vtt - YMM4セリフ抽出 WebVTT変換ツール

「ゆっくりMovieMaker4 (YMM4)」のプロジェクトファイル（`.ymmp`）からセリフを抽出し、動画字幕規格である **WebVTT (.vtt)** 形式に変換するWebツールです。

## 🚀 技術仕様
- **依存関係**: なし（バニラJavaScript / 標準ブラウザAPIのみで動作）
- **プライバシー**: サーバー通信なし。ブラウザ上で全ての処理を完結するため、プロジェクトファイルが流出することはありません。
- **対応形式**: YMM4ボイスアイテム（VoiceItem）のセリフ抽出に対応。

## 🛠 使い方
1. YMM4で保存した `.ymmp` ファイルを本ツールに読み込ませます。
2. プロジェクトのFPSに基づき、各セリフの開始・終了時刻が自動計算されます。
3. 生成されたVTTファイルをダウンロードし、動画プレイヤーや編集ソフトの字幕トラックとして利用してください。
