# High-Speed Large Data Visualizer

10万件規模のテキストデータ（数値データ）を、ブラウザ上で一瞬でグラフ化するHTMLツールです。
WebGL（Plotly.js）を利用しているため、大量のデータ点があってもマウスホイールで快適に拡大・縮小・移動が可能です。

## 🚀 特徴

- **超高速描画**: 10万点以上のデータでも、GPU（WebGL）を使用してスムーズに描画。
- **直感的な操作**: マウスホイールでズーム、ドラッグで移動、ダブルクリックでリセット。
- **インストール不要**: HTMLファイル一つで動作するため、サーバーやPython環境などの構築が不要です。
- **自動スケーリング**: アップロードしたデータの値に合わせて、自動的に最適な倍率で表示されます。

## 🛠 使い方

1. [ページ](https://github.com/anesewo/tools/graph.html)を開きます。
2. `ファイルを選択` ボタンから、1行に1つの数値が書かれたテキストファイル（.txt または .csv）を選択します。
3. `グラフを描画` をクリックすると、即座にグラフが表示されます。

## 🖱 グラフの操作方法

| 操作 | 動作 |
| :--- | :--- |
| **マウスホイール** | カーソル位置を中心に拡大・縮小 |
| **左ドラッグ** | グラフの表示範囲を移動（パン） |
| **ダブルクリック** | 全体表示（オートスケール）に戻す |
| **右上のカメラアイコン** | 現在のグラフを画像として保存 |
