# SVG Batch Outline Generator

TTF/OTFフォントを読み込み、複数のテキストを一括でSVGパス（図形）に変換して保存するブラウザベースのツールです。

## 特徴
- 文字をパスに変換するため、フォント未インストール環境でも表示可能
- 画面サイズに合わせた配置設定（左・中央・右・中央開始）
- ZIP圧縮なしのフォルダ直接保存（ウイルス誤検知対策）
- ファイル名のプレフィックス（接頭辞）指定可能

## 動作環境
- Google Chrome (推奨)
- Microsoft Edge  

※ `File System Access API` を使用しているため、上記以外のブラウザやモバイル版では動作しません。

## 使い方
1. [ページ](https://github.com/anesewo/tools/svg_generator)をブラウザで開きます。
2. フォントファイル(.ttf / .otf)を選択します。
3. 地名などのテキストを1行ずつ入力します。
4. 配置やファイル名を設定し、「保存を開始」をクリックして出力先フォルダを選択してください。
